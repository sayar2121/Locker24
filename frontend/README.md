# Locker 24

A premium, secure personal document vault built with React and Tailwind CSS.

## Features
- **Secure Auth**: Modern login and registration flows.
- **Premium UI**: Glassmorphism design, smooth animations, and responsive layout.
- **Document Management**: Grid and list views for your personal documents.
- **Dark Mode**: Fully integrated dark and light themes.

## Getting Started

1. **Install Dependencies**:
   ```bash
   cd frontend
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Tech Stack
- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router 7
